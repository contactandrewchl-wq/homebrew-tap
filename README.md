# Homebrew tap de Turtle 🐢

Instalá [Turtle](https://github.com/contactandrewchl-wq/turtle-mcp) —memoria persistente y
coordinación multi-agente para CLIs de IA— con Homebrew en **macOS** (y Linux/Linuxbrew):

```sh
brew install contactandrewchl-wq/tap/turtle
```

O en dos pasos:

```sh
brew tap contactandrewchl-wq/tap
brew install turtle
```

Después:

```sh
turtle --version
turtle setup claude-code   # o: turtle setup codex
```

La formula descarga el binario ya compilado del último
[Release](https://github.com/contactandrewchl-wq/turtle-mcp/releases) (no compila nada).
